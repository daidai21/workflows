# WorkFlows

workflows with C++11.

### Core Features

* [ ] 支持工作流种类： 条件流、顺序流、重复流、并行流
* [ ] 支持定义工作流方式： yml、json、C++代码
* [ ] 支持不同种类流叠加
* [ ] 支持持久化存储
